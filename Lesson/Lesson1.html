Input & Output

當我們可以從螢幕上看到Hello World表示我們已經將C++的程式印到螢幕上了～


其實這就是C++的Output，所以其實很簡單，想要將C++程式計算出來的答案印出來在螢幕上的話，只要在程式中打上cout，意思電腦Output


那為甚麼只要打上cout就會將C++ Output到螢幕上面ㄋ，原因在前面那章節應該有誰提到過，也就是因為程式開頭有打上#include這會讓程式知道要從iostream的函式庫中找到cout



這時後電腦就知道該怎麼做了！！！！



接下來要介紹的是cin，它是一個類似cout，只是一個是cin是輸入，而cout 是輸出，所以就可以寫出類似下面的程式。



#include
using namesapcese std;
int main(void)
{
cout << "請輸入你的名字：";
cin >> name;
cout << "Hello " << name << endl;
}


好像蠻理想的，就是電腦輸出要使用者輸入名字，而使用者輸入，電腦就會輸出Hello 名字，但是實質上不是，會發現輸入進去，程式會出現一些訊息，並不是Hello甚麼ㄉ，而是error : 6 : name.........相信看到這，你已經成功一半了，但是為甚麼是一半？


其實是因為電腦根本不知道name是甚麼，name是我自己定義的一個變數，就像我自己去找一個盒子來裝使用者輸入的東西，但是電腦看不懂，所以其實電腦真的很笨的。


那你便會問我說，該怎麼辦，其實很簡單，你只要多一個步驟告訴電腦，現在你自己找了一個盒子，這個盒子裡面會裝字串，我們稱為「宣告」要加在那裡ㄋ，必須在main function 中，並且在cin >> name;之前。
