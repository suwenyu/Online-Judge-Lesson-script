Array

接下來要進入新的觀念ㄌ，這被稱為陣列Array。
它的特性是甚麼ㄋ？其實它就只是一個比較長的記憶體空間，可以讓你儲存更多同類型的資料，比如說，你可能會有100筆客戶的資料，這時候你當然就會想要用一個array把它存起來。這樣也比較好管理，比較好做資料的處理。


這時候你就會問，那陣列到底要怎麼用，其時很簡單。
一樣要宣告，例如：int name_of_array[100];，這樣就可以存100個integer的資料ㄌ，當然也可以有其他不同的類型，像：float、char......等等ㄉ。



接下來你就會問，那如果我有100筆資料，我要怎麼放進array裡面，好像沒辦法直接cin對不對？沒錯！！！不能直接cin，那怎麼辦？
你只要想array既然是放同類型的東西，要重複放置這些東西，所以很直覺要重複做這些動作是不是應該要for loop，答案就出來ㄌ。


for (int i = 0 ; i < length(array_name) ; i++ )
{
cin >> array_name[i];
}
我解釋一下吧，所以我用一個for迴圈把array從第0個位置開始放，放到array的長度滿為止，然後一直重複執行cin這件事情。
這樣每一筆資料就會被放到array的每個位置中ㄌ。
