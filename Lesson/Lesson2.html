For Scope

介紹完cout、cin 接下來我要介紹的東西可以幫助我們省去很多重複的事情。那就是for 迴圈


for在甚麼時候用ㄋ？答案很簡單，當我們想要做一件事情很多次，一直重複做，這時候我們一定不會想一直寫重複的東西，所以就可以用for迴圈解決。



舉個例子來說吧：假設我要從1加到100，當然可以慢慢用打得從1加到100，這樣可能會打很久耶～那所以這時候我們就會想到要用for，那該怎麼寫ㄋ？很簡單～

int sum;
for(int i = 1 ; i <=100 ; i++)
{
sum += i ;
}



好，那應該是需要我解釋一下每一個位置，for()中需要擺放的東西，三個位置

  1. 變數值，在for迴圈中可能會用到的變數在這裡宣告
  2. for迴圈的終止條件，當符合的時候就會跳離迴圈
  3. 最後是變量，一次要移動多少的變量


相信大家應該都大致了解了，for就可以有很大的應用範圍，如果都不寫條件
for(;;)的效果便是無窮迴圈，這時候如果要暫停程式可以按pause sysqr來暫停
