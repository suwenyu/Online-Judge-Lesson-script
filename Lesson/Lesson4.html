Function

接下來要介紹的是也是在程式語言中重要的觀念，也就是Function，其實一直都有再用function只是你沒有特別感受到。


我們一般而言在寫main的時候就是在寫function，我們會呼叫一個int main(){}就是寫寫一個function，那function是甚麼ㄋ？


我先來說說函式的特性，為甚麼要用到函式，因為有些重複的動作如果沒次都要再重寫會很浪費時間、空間，比如說我要對很多變數做平方、或是等等一樣的動作。


這時，如果我有寫一個函式負責的工作就是平方，這樣會節省很多時間，也不用很麻煩每次都要一直寫。例子如下：


int square( int a )
{
a = pow(a,2);
}


函式有一些觀念很重要，函式有傳入傳出值，各有各的型態，可能傳入是integer，而回傳是string都有可能，所以要分別定義，要怎麼定義ㄋ？
很簡單，function name前面的是回傳的型態，在括號中的則是傳入函式的型態 很重要喔～要記起來～之後可能有更多的型態必須處理。



函式還有一些比較特別的應用，例如說；遞迴，遞迴的意思就是利用函式本身呼叫自己，讓它循環的一直找到一個解答再往回推～，現在因為functional programming火紅，所以遞迴越來越重要。


最直接的例子，用遞迴計算6!～聽起來很難～但是其實不會，想想，我要讓函式呼叫自己本身嘛！然後6!= 6*5*4*3*2*1，可以用iterative很值觀的寫法


for(int i = 6 ; i>= 1 ;i -- )
sum *= i;
這是最直觀ㄉ～但是我們現在在要來看的是遞迴的寫法，遞迴很重要的兩個觀念
1. 函式呼叫自己
2. 要有終止值


int factorial ( int a )
{
retrun answer * factorial ( a-1 ); }
