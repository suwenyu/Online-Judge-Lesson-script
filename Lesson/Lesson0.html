Say Hello to C++

看過不少程式語言，我覺得程式基礎還是要打在C++上，因為C++有助於我們學習很多的解題方式、演算法等等的，也可以快速上手程式語言的邏輯，所以我想以C++當作教學語言。


要寫程式首先需要有編輯器，最簡單上手的便是記事本，當然，很不好用，那要用什麼ㄋ？？
我自己剛開始練習是使用dev C++真的蠻容易上手的，不過CodeBlock也是值得推薦。下載下來安裝完成就可以使用嘍！！以下附上連結：
dev C++ CodeBlock
安裝好之後，就是準備開始我們的第一支程式吧！！！
首先要先看到C++的一些規定：
#include
using namesapce std;
int main (void)
{
//your code
return 0;
}
可以看到這樣一個簡單的程式，其實什麼都不會輸出，但是對於電腦卻是可以看得懂的喔！！！
我需要一行一行稍微的帶過一下，讓大家了一下為什麼需要這些英文。


#include -> 的工作是讓電腦在編譯的過程會把一些已經寫好ㄉ檔案加入進來，像iostream就是之前的人已經寫的一個I/O檔，所以在使用cout cin 等等指令，電腦它才看得懂。至於編譯是什麼，之後會在提到。

using namespace std; -> 這行很重要，因為如果沒有這行，你就必須告訴電腦cout cin 是從哪個libarary出現的，比如說：std::cout std::cin 因為cout cin 是在std 這個library你如果沒有告訴電腦，它可能會找不到。

int main (void) -> 這行便是主程式，也就是程式的起頭，可以想像成一個起點，所有的執行都是從這邊開始ㄉ，上面都只是在告訴電腦我需要用到什麼工具這樣而已。

{ -> 括弧中間放要寫的程式
// your code -> 寫上自己的程式
return 0; -> 可有可無，只是告訴電腦程式結束了！！

}


C++有一個既定的規定，每一行程式的結束都要加上;不然電腦會不知道什麼時候是看完一句話。


說了這麼多，最重要的還是練習呀！！！來看看習題吧！！
